## Claude Code 更新情報（v2.0.73時点）

### 新機能

| 機能 | 説明 | バージョン |
|-----|------|----------|
| Chrome統合（Beta） | ブラウザ直接制御、`claude.ai/chrome` | v2.0.72 |
| セッション名前付け | `/rename`, `/resume <name>` | v2.0.73 |
| Plugin System | マーケットプレイス、`/plugin` コマンド | v2.0.70+ |
| 背景エージェント | バックグラウンド実行 | v2.0.60 |
| Plan Mode強化 | Plan Subagent、より正確な計画 | v2.0.64 |
| Opus 4.5対応 | Pro ユーザー利用可能 | v2.0.51 |

### Hooks拡張（10種類）

| イベント | 説明 |
|---------|------|
| PreToolUse | ツール実行前（ブロック可能） |
| PostToolUse | ツール実行後 |
| PermissionRequest | 権限ダイアログ表示時（自動承認/拒否） |
| UserPromptSubmit | ユーザープロンプト送信時 |
| Notification | 通知イベント |
| Stop | Claude Code応答完了時 |
| SubagentStop | サブエージェント完了時 |
| PreCompact | コンパクト処理前 |
| SessionStart | セッション開始/再開時 |
| SessionEnd | セッション終了時 |

### Skills拡張
- `skills` frontmatter対応（サブエージェント用スキル自動ロード）
- Anthropic管理Skills（PowerPoint, Excel, Word, PDF）

### MCP対応強化
- ワイルドカード対応: `mcp_server_*` で全ツール許可/拒否
- `/mcp enable/disable` コマンド
- OAuth（Slack等）サポート改善

### 新CLIフラグ
```bash
claude --resume <name>           # 名前付きセッション再開
claude --session-id custom-id    # カスタムセッションID
claude --fork-session            # セッションフォーク
claude --agent agent-name        # エージェント指定
```
