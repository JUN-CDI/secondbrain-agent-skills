# 🎯 Cursorレイアウト設定 - ウルトラワイド版

**対象:** ウルトラワイドディスプレイ（21:9以上）を使用している場合

---

## 📋 レイアウトオプション

### オプション1: 上下分割（標準版）

```
┌─────────────────────────────────────────────────────┐
│  Explorer (左)  │  Editor (中央)  │  Terminal (右)  │
│                 │                  │                 │
│                 │                  │  claude         │
│                 │                  │  ─────────────  │
│                 │                  │  codex         │
└─────────────────────────────────────────────────────┘
```

**特徴:**
- ✅ 標準的なレイアウト
- ✅ ターミナルが縦に長い（ログを見やすい）
- ✅ エディタエリアが広い

---

### オプション2: 左右分割（ウルトラワイド推奨）

```
┌─────────────────────────────────────────────────────────────┐
│  Explorer (左)  │  Editor (中央)  │  claude  │  codex      │
│                 │                  │          │             │
│                 │                  │          │             │
│                 │                  │          │             │
│                 │                  │          │             │
└─────────────────────────────────────────────────────────────┘
```

**特徴:**
- ✅ ウルトラワイドに最適化
- ✅ 2つのターミナルを同時に見やすい
- ✅ エディタエリアも十分に広い
- ✅ 横方向の情報量が多い

---

## 🚀 ウルトラワイド版のセットアップ手順

### Step 1: 基本レイアウトの設定

1. **左側Explorerを表示**
   - `Cmd+B` でサイドバーを表示
   - Explorerを選択

2. **ターミナルパネルを右側に移動**
   - ターミナルを開く: `Cmd+Shift+\``
   - タイトルバーをドラッグして右端に移動

### Step 2: ターミナルを左右に分割

1. **最初のターミナルを作成**
   - 右側のターミナルパネルで新規ターミナル作成
   - タブを右クリック → `Rename` → `claude`

2. **ターミナルを左右に分割**
   - ターミナルタブを右クリック → `Split Terminal` → `Split Right`
   - または: `Cmd+\` を押してから、分割方向を選択

3. **2つ目のターミナルをリネーム**
   - 右側のターミナルタブを右クリック → `Rename` → `codex`

### Step 3: レイアウトの調整

1. **ターミナルの幅を調整**
   - ターミナル間の境界線をドラッグして幅を調整
   - claudeとcodexの幅を均等にするか、用途に応じて調整

2. **エディタエリアの確認**
   - 中央のエディタエリアが十分な幅があることを確認
   - 必要に応じてターミナルの幅を調整

## 🔧 tasks.jsonの設定（ウルトラワイド版）

現在の設定は上下分割用ですが、左右分割でも動作します。

**現在の設定:**
```json
{
  "presentation": {
    "group": "ai-terminals",  // 同じグループで分割表示
    ...
  }
}
```

**左右分割の場合:**
- `group` を同じにすることで、自動的に分割表示されます
- 分割方向は、Cursorが自動的に最適な方向を選択します
- 手動で調整する場合は、ターミナルタブを右クリック → `Split Right` を選択

---

## 💡 ウルトラワイド版のメリット

### 上下分割 vs 左右分割

| 項目 | 上下分割 | 左右分割 |
|------|---------|---------|
| **ログの見やすさ** | ✅ 縦に長い（スクロールしやすい） | ⚠️ 横に広い（1行が長い） |
| **2つのターミナル比較** | ⚠️ 上下に分かれる | ✅ 並べて見やすい |
| **エディタエリア** | ✅ 広い | ✅ 広い（ウルトラワイドなら十分） |
| **推奨用途** | 標準ディスプレイ | ウルトラワイドディスプレイ |

---

## 🎯 推奨レイアウト

### ウルトラワイド（21:9以上）の場合

**推奨: 左右分割**
- claude と codex を左右に配置
- 2つのターミナルを同時に比較しやすい
- エディタエリアも十分に広い

### 標準ディスプレイ（16:9）の場合

**推奨: 上下分割**
- claude と codex を上下に配置
- ログを見やすい
- エディタエリアを最大限に活用

---

## 📝 設定の切り替え方法

### 上下分割から左右分割に変更

1. **現在のターミナルを閉じる**
   - 各ターミナルタブを右クリック → `Kill Terminal`

2. **新規ターミナルを作成**
   - `Cmd+Shift+\`` で新規ターミナル作成
   - リネーム: `claude`

3. **左右に分割**
   - ターミナルタブを右クリック → `Split Terminal` → `Split Right`
   - または: `Cmd+\` を押してから、分割方向を選択

4. **2つ目のターミナルをリネーム**
   - `codex` とリネーム
