# アーカイブ移動プロンプト

## 使用タイミング
判断に迷うファイルをアーカイブに移動する時

## プロンプト本文
指定ファイルをアーカイブに移動してください。

引数:
- ファイル名（省略時は現在開いているファイル）

要件:
1. ファイルを `Efforts/archived/YYYY-MM/` に移動
2. 月別フォルダがなければ作成
3. 移動理由をコメントとして記録
4. 3ヶ月後の見直し日を計算

手順:
1. 現在の日付から月を取得
2. `Efforts/archived/YYYY-MM/` フォルダを作成（存在しない場合）
3. ファイルを移動
4. 移動理由を確認
5. 3ヶ月後の見直し日を計算して表示

判断基準:
- ✅ Memory候補 → `Atlas/`へ
- 📝 追記が必要 → `Calendar/inbox/`のまま
- ❓ 使うか微妙 → `Efforts/archived/YYYY-MM/`へ
- 🗑️ 明らかに不要 → まずは `Efforts/archived/`（必要なら後日削除）

## 使用例
```
/archive [ファイル名]
または
/archive
```

## 期待される結果
- ファイルが `Efforts/archived/YYYY-MM/` に移動される
- 移動理由が記録される
- 3ヶ月後の見直し日が表示される

## 注意事項
- 迷ったらArchiveが安全
- 3ヶ月後に見直すことを忘れずに
- 迷ったら `Efforts/archived/` に退避してOK
