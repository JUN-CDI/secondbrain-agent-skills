# Atlas Note作成支援プロンプト（旧: Memory Note）

## 使用タイミング
`Calendar/inbox/` のメモを、`Atlas/` の「長期で参照するノート」に昇華したい時

## プロンプト本文
指定された inbox メモを、`Atlas/` のノートとして整えてください。

引数:
- ファイル名: 変換元のファイル（ユーザーが指定）

実行内容:
1. 指定ファイルの内容を読み込む
2. 要点を抽出し、1ノート1テーマに整形する（必要なら分割案も提示）
3. frontmatter（title/tags/created）を最小で付与
4. 関連リンク候補を1〜2個提案
5. ユーザー承認後、`Atlas/` に移動（フラット運用: サブフォルダは作らない）
6. MOCへ1リンク追加（最低1つ）
   - `Atlas/🗺️ Atlas-MOC.md`
   - and/or `Atlas/_AI-MOC.md` など

## 使用例
```
/make-memory 2024-10-22-API設計メモ.md
```

## 期待される結果
- 内容の評価（改善点）
- Atlas用に整形されたノート案
- 承認後、`Atlas/` へ移動 + MOC更新

## 注意事項
- 3ヶ月後も参照する価値があるかを必ず確認
- 迷う場合は無理にAtlasに入れず `Calendar/inbox/` に残してOK
