# inboxレビュープロンプト（旧: 03_Input）

## 使用タイミング
週1回（日曜夜）に `Calendar/inbox/` を分析して「整理案だけ」提示したい時

## プロンプト本文
`Calendar/inbox/` を分析して、整理案を提示してください（この段階では移動しない）。

実行内容:
1. `Calendar/inbox/` 内の全ファイルをスキャン
2. 各ノートを以下の基準で評価し、理由を添える
   - Atlas候補（3ヶ月後も価値あり）
   - Projects候補（期限あり）
   - Areas候補（継続活動）
   - inboxに残す（今週使う/迷う）
   - archived候補（使い終わり）

評価基準（例）:
- 自分の言葉で書けているか
- 1ノート1テーマに寄っているか
- 完結性
- 最小限のリンク/タグ

## 使用例
```
/inbox-review
```

## 期待される結果
- inbox棚卸しレポートが生成される
- 各ファイルの提案先と理由が表示される
