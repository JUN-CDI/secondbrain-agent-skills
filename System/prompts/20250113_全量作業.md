# 全量作業プロンプト

## 使用タイミング
もれなく全てステップバイステップで作業する時

## プロンプト本文
もれなく全てステップバイステップで作業して

または、より具体的に：

以下の作業を漏れなく全てステップバイステップで実行してください：
1. [ステップ1]
2. [ステップ2]
3. [ステップ3]
...

各ステップの完了を確認してから次に進んでください。

## 使用例
```
もれなく全てステップバイステップで作業して
```

または

```
以下の作業を漏れなく全て実行してください：
1. Calendar/inbox内の全ファイルを読み込む
2. 各ファイルの内容を分析
3. Atlas / Efforts / Efforts/archived のどれに行くべきか提案
4. 必要なら移動を実行（または提案だけにする）
```

## 期待される結果
- 全てのステップが漏れなく実行される
- 各ステップの完了が確認される
- 最終的な結果がまとめられる

## 注意事項
- ステップを飛ばさない
- 各ステップの完了を確認
- エラーが発生した場合は報告
