# inbox整理・振り分けプロンプト（旧: 03_Input）

## 最小運用（これだけ）
- 日曜夜に `/organize-input` を1回回す（迷ったら `Calendar/inbox/` のまま or `Efforts/archived/`）

## 使用タイミング
週1回（日曜夜）に `Calendar/inbox/` 内のファイルを整理して移動したい時

## プロンプト本文
`Calendar/inbox/` 内のファイルを内容に基づいて適切なフォルダに振り分けてください。

**重要**: 全てのファイルの中身を必ず読み込んで、内容を丁寧に分析・理解してから振り分けます。

### 振り分け先（ACE）
- 長期知識（3ヶ月後も参照）→ `Atlas/`（フラット運用）
- 今週〜今月使う / 迷う → `Calendar/inbox/`（残す）
- 期限あり/進行中（プロジェクト） → `Efforts/`（必要なら `Efforts/<project>/` を作る）
- 手順書/ナレッジ（再利用） → `Atlas/`
- 完了/不要/保管 → `Efforts/archived/`

### 移動後にやること（最小）
- frontmatterの `tags` を必要最小限で付与
- 重要概念があれば `[[links]]` を1〜2個
- `Atlas/` に移した場合はMOCへ1リンク追加（最低1つ）
  - `Atlas/🗺️ Atlas-MOC.md`
  - and/or `Atlas/_AI-MOC.md` など

## 使用例
```
/organize-input
```

## 期待される結果
- ファイルが適切なフォルダに移動される（移動サマリ付き）
- `Atlas/` に移したものはMOC更新が案内される

## 注意事項
- **最重要**: ファイル名だけで判断せず、必ず全文を読んでから判断
- 迷ったら無理に動かさず `Calendar/inbox/` のままでOK
