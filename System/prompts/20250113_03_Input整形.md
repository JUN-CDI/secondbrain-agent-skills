# 03_Input整形プロンプト

## 使用タイミング
`Calendar/inbox/` の“雑メモ”を、検索しやすい「1ノート」に整えたい時（移動はしない）。

## プロンプト本文
次のノートを、`Calendar/inbox` 内で「使える1ノート」に整形してください。

要件:
- 内容を必ず全文読んで理解する（ファイル名だけで判断しない）
- 1ノート1テーマに寄せる（複数テーマなら分割案も提示）
- 先頭に簡単な frontmatter（title/tags/created）を付ける（重くしない）
- 1〜3個だけ関連リンク候補を提案（貼りすぎない）
- “次アクション”を1つだけ書く（なければ空でもOK）

出力（整形後の例）:
```markdown
---
title: タイトル
tags: [input]
created: YYYY-MM-DD
---

# タイトル

## 概要
1〜2文

## 要点
- ...

## 次アクション
- [ ] ...
```

注意:
- ここではフォルダ移動しない（移動は週次でOK）
- 移動までやるなら `/organize-input` か [[20250113_03_Input整理]]
